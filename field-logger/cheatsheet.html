Field Logger — Field Cheat Sheet

Tool 46 · camera-mounted phone · tap riders → field_log.json → Auto-sort
THE RHYTHM: tap the rider → shoot → tap next ▼

1 Before the class (once)

  • Build the list: Tool 46 → B → paste the Equipe class id / startlists URL.
  • Open on phone: with PC → Tool 46 → S, open the http://<pc-ip>:8137/ it prints (same Wi-Fi). No PC → open the Pages URL (below). Then Add to Home screen.

2 Mount (sync only if a clock is off)

  • Mount the phone on the camera.
  • Clocks set right? Skip Sync. The 1DX3 GPS-syncs its clock and geotags photos; just keep the FX6 clock set. Ingest then uses phone time = camera time.
  • Only if a clock might be off: tap Sync → photograph the QR, or type the camera's clock (pick FX6 / 1DX3).

3 While shooting

  • Set the activity once (e.g. Round) — it carries to each rider.
  • Tap the rider as they enter (or ▼ Next). The green NEXT line shows who's up + ETA; ★ in the list = priority (Swedish + top-3).
  • Tap a shot chip when you grab one: Wide · Tight · Detail · React · Follow.
  • ★ Flag a keeper — then an optional one-tap why (hero / fall / win / …) · 🎤 Note to say what a special clip was.
  • Tap the status line → Coverage: chase shot gaps, the priority riders with no shot yet, and tick the b-roll checklist.
  • ⤢ big mode + the haptic buzz = log without looking away; the FX6/1DX3 pill tags the active camera.

4 End of the card

  • Wrap: tap 📋 for the card summary (riders / flags / coverage + priority gaps), then…
  • To the PC: tap Send (LAN serve → lands in the project) or Export (downloads).
  • Ingest: Tool 46 → I → pick the camera → writes field_log.json.
  • Sort: run Auto-sort (Tool 22) — it picks up the field log and routes everything.

★ Button legend

Activities — Round / Warm-up / Walk / B-roll / Portrait / Interview / Ceremony / Scenery / Stable
Shot chips — Wide / Tight / Detail / React / Follow (coverage)
NEXT line — who's up next + ETA (predictive)
★ in list — priority rider (Swedish + top-3) to cover
★ Flag — keeper, + optional one-tap reason
🎤 Note — record a timestamped voice note
FX6 / 1DX3 — active camera (tap to switch)
⤢ Big mode — giant card, eyes-free (haptics on)
📋 Summary — end-of-card counts + gaps
Sync — clock sync + Tag venue GPS
Send / Export — push to PC / download the session

! If you skip the logger

Everything still sorts — the logger is just the most certain layer. Auto-sort falls back per file:
Photos: field-log → bib OCR (-UseBib) → timestamp window → _Loose.
Clips: field-log → scene/slate → name in filename → voice (Whisper) → timestamp; b-roll → _<Type> folder.
Caveat: timestamp sorting needs the camera clock ≈ real time — the 1DX3 GPS-syncs this automatically; just keep the FX6 clock set (or Sync once if unsure).
